Requirements

- Elasticsearch
- logstash
- kibana
- Spring Boot 

## Installation process
- Download and unzip required file
- add the log file path on application.yml inside your Spring Boot application
  ```
  logging:
  file:
    name: /home/user/log/xxx.log
  ```
